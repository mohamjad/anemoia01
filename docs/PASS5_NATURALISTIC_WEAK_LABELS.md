# Pass 5 Naturalistic Weak Labels

Pass 5 adds the first executable scaffold for AJILE12-style naturalistic weak
labels:

- naturalistic behavior proxy events
- confidence-weighted weak target distributions
- JSONL event IO
- proxy confidence and ambiguity summaries
- naturalistic weak-label protocol results
- Markdown report rendering
- CLI evaluation from event and prediction JSONL files

This is still a scaffold. Behavioral outputs and timing windows are treated as
intent proxies, and confounding remains explicit in the report language. No
claim is made that these events reveal true intent directly.
